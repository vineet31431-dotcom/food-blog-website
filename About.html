Our story

Caparico is a responsive food website project created to demonstrate strong front-end web development skills. It highlights expertise in HTML, CSS, and JavaScript, including features like dark mode, a responsive navigation bar, and optimized mobile layouts.

About the creator

Hi! I’m Vineet, a passionate learner building a career in web development and UI/UX design. This project was built as part of my portfolio to showcase my ability to design and develop functional, visually appealing websites.

Technologies Used

This project was built with HTML5, CSS3, JavaScript, and responsive design principles to ensure fast loading times and seamless user experiences on all devices. Resources used: icons by Font Awesome, images from Unsplash, and content research with ChatGPT.
